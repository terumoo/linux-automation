# Disk Usage Display Script

## Overview
`disk_check.sh` is a **basic Linux Bash script** that displays disk usage information for all mounted filesystems using standard Linux utilities.  
The script serves as an introductory automation exercise and a foundation for more advanced system monitoring tasks.

---

## Features
- Displays disk usage in a human-readable format  
- Compares usage against a configurable threshold
- Uses standard Linux tools available on most distributions  

---

## Script Usage

### Make the script executable
```bash
chmod +x disk_check.sh

```
### Run the script
```bash
./disk_check.sh
